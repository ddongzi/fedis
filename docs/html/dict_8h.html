◆ dictDelete()

int dictDelete ( dict *  dict,
const void *  key 
)

Deletes a key-value pair from the dictionary.

Parameters
dictThe dictionary from which to delete the key-value pair.
keyThe key of the key-value pair to delete.
Returns
DICT_OK if the key-value pair was successfully deleted, DICT_ERR otherwise.
Note
This function checks if the dictionary and key are not NULL, triggers an expansion if necessary, and performs a deletion if the key is found in the dictionary. If the dictionary is currently rehashing, the function triggers a rehash step before performing the deletion. If the key is not found, the function returns DICT_ERR.
